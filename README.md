# Game of Life

## Description

This repository contains an implementation of Conway's Game of Life, developed in Python. Conway's Game of Life is a classical cellular automaton devised by mathematician John Horton Conway. 
The simulation consists of a grid of cells, each of which can be alive or dead. Cells evolve at each generation based on a set of simple rules derived from the states of neighboring cells. 
This project provides a simple, configurable framework to visualize and experiment with the Game of Life in Python.

## Features

- 100% Python implementation
- Customizable initial patterns
- Simple, readable code structure
- Easily extensible for additional visualization or interactivity

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/kaemiks/Game-Of-Life.git
   cd Game-Of-Life
   ```

2. (Optional) Install any required dependencies:
   ```bash
   pip install -r requirements.txt
   ```

## Usage

To run the simulation, execute:
```bash
python game_of_life.py
```
*(Replace `game_of_life.py` with the correct entry point if different.)*
